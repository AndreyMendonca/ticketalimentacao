# 🎟️ Gerador de Tickets (SENAI / SESI)

Aplicação web desenvolvida em Next.js para geração de tickets em PDF, utilizada para impressão de tickets institucionais (SENAI e SESI).

## 🚀 Funcionalidades

- Seleção do tipo de ticket:
  - SENAI
  - SESI
- Escolha de data personalizada
- Geração automática de 18 tickets por página
- Exportação em PDF com nome dinâmico
- Layout pronto para impressão
- Responsivo (funciona em desktop e mobile)
